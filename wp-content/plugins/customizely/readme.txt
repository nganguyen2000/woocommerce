== Description ==
Using **Customizely** anyone can add custom controls to the customizer with easy to use drag and drop option builder. It can full fill your WordPress Customizer custom controls desire very easily. It’s simple to use user interface will help you to build options very easily. **Customizely** does not only add options to WordPress Customizer, but It also renders CSS for you. It supports 2 more responsive devices than default WordPress. Just try it and you will love it.

= Why you should use Customizely? =
Every theme doesn’t provide every option to customize your site. Contacting support to request a new option for the theme, it’s a long term process. Or you should hire a developer. It's cost-effective. That’s why we built **Customizely**, fully drag and drop WordPress Customizer options builder. It will save money and time both. It comes with drag and drop builder, auto-generated CSS output, and 13 Customizer custom controls with responsive features.

= Drag and Drop. No coding knowledge required. =
Fully drag and drop WordPress Customizer options builder with a simple user interface. Anyone can build an option without any coding knowledge. It’s very easy to use.

= Auto-generated CSS output. =
WordPress Customizer brings the live editing feature. And **Customizely** extends this feature with auto-generated CSS. Every option can set for CSS output by just adding CSS selector and property. You can see every change in the live view.

= Responsive. Mobile First Editing. =
All controls have a mobile responsive feature. Turn one it with CSS and enjoy live design editing with 5 devices option. WordPress Customizer has 3 devices (Desktop, Tablet, and Mobile) by default. **Customizely** extend it to 5 devices (Desktop, Laptop, Tablet, Mobile Landscape, and Mobile).

= Controls =
**Customizely** comes with 11 controls and more controls are coming soon.

* Text
* URL
* Textarea
* Checkbox
* Radio
* Select
* Switch
* Range
* Number
* Icon
* Color
* Media
* Image Select

**Customizely** also provides API for developers to add WordPress Customizer custom controls from there theme or plugin. Theme developer can use to build their theme options and give their customers to get the advantage of the drag and drop option builder.

== Installation ==

1. Upload the entire `customizely` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Can I manage existing options? =
If the option is not build with drag and drop feature of this plugin, no you can't manage those options from option builder.
